💡 Utilizare NPM Scripts:

  • npm install - Instalează dependencies (sass)
  • npm run sass - Compilează menu.scss → menu.css
  • npm run sass:watch - Watch mode (auto-compile la modificări)

📐 Formule Tehnice pentru Sisteme Satelit

Calcule avansate pentru optimizarea transmisiei satelit

Formula Shannon-Hartley (Capacitatea Canalului)

C = B × log 2 1 + S N

unde: C = capacitate (bits/s), B = lățime bandă (Hz), S/N = raport semnal/zgomot

Câștigul Antenei Parabolice

G = η · π · D λ 2

unde: G = câștig antenă, η = eficiență, D = diametru, λ = lungime undă

Link Budget pentru Transmisie Satelit

P r = P t + G t + G r − 20 · log 10 4 π d λ

Link Budget - calculul puterii primite la recepție: Pr = putere recepție (dBm), Pt = putere transmisie (dBm), Gt,r = câștiguri antene, d = distanță satelit

Matrice Parametri Sistem

Frecvență = 11.7 − 12.75 GHz Polarizare = Circulară / Liniară EIRP = 52 − 56 dBW π ≈ 3.14159265

Toate formulele sunt utilizate în optimizarea sistemelor noastre de recepție satelit profesionale.
