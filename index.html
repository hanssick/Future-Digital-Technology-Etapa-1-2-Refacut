Ecuația lui Friis — Puterea Recepționată

Pr = Pt · Gt · Gr · λ 4πd 2

Pr = putere recepție, Pt = putere transmisie, Gt,r = câștiguri antene, λ = lungime undă, d = distanță satelit

Câștigul Antenei Parabolice

G = η · π·D λ 2

G = câștig antenă, η = eficiență, D = diametru, λ = lungime undă
